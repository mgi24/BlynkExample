Super simple blynk example using ESP32
How to use:

1. Install this library <img width="355" height="345" alt="image" src="https://github.com/user-attachments/assets/429b83e3-46b5-41c2-bd84-ecc2b9ff6c97" />

2. login to https://blynk.cloud/ and make a template <img width="1118" height="558" alt="image" src="https://github.com/user-attachments/assets/2d304517-4712-4121-89d4-33f5d9793eed" />

3. Copy the template ID to code <img width="1875" height="583" alt="image" src="https://github.com/user-attachments/assets/c097f676-ca48-4436-86e3-8e0feaef6db2" />

4. Create these datastreams inside the template <img width="1895" height="617" alt="image" src="https://github.com/user-attachments/assets/3284e41f-8dea-4b57-8098-bc35878d2834" />

5. Create these simple dashboard <img width="1355" height="859" alt="image" src="https://github.com/user-attachments/assets/ba0c1b26-3274-4b69-9ee7-cd4586a7e50e" />

6. Goto devices and add a device, copy AUTH token to code <img width="1903" height="547" alt="image" src="https://github.com/user-attachments/assets/683c9703-3ffe-495d-876c-57c4599cfad0" />

7. Add wifi and password and youre done!
